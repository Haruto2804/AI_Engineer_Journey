## ⚡ Các Dự Án Thực Chiến Nổi Bật

### 1. 🏦 AI Chấm Điểm Tín Dụng (Credit Scoring)
> File: `ProjectThucHanhCacKienThucDaHoc/AI-Credit-Scoring-System/`
- **Mô tả:** Hệ thống AI tự động duyệt/từ chối hồ sơ vay vốn
- **Kỹ thuật:** Decision Tree, Train/Test Split, Confusion Matrix
- **Kết quả:** AI đạt độ chính xác > 80% trên tập kiểm tra

### 2. 🏠 AI Định Giá Bất Động Sản (Bài 16)
- **Mô tả:** Dự đoán giá nhà dựa trên diện tích & số phòng ngủ
- **Kỹ thuật:** RandomForestRegressor, MAE, R² Score

### 3. 🚗 AI Định Giá Ô Tô Cũ (Bài 16-17)
- **Mô tả:** Định giá xe đã qua sử dụng dựa trên năm SX & số km
- **Kỹ thuật:** Regression, Joblib Deployment → File `.pkl`

### 4. 👥 AI Giữ Chân Nhân Tài (Bài 15)
- **Mô tả:** Dự đoán nhân viên nào sắp nghỉ việc
- **Kỹ thuật:** RandomForest + GridSearchCV + Feature Importance
- **Insight:** Yếu tố "Hài lòng với Sếp" ảnh hưởng lớn nhất!

### 5. 🩺 AI Bác Sĩ Tim Mạch (Bài 13)
- **Mô tả:** Chẩn đoán bệnh tim dựa trên tuổi & huyết áp
- **Kỹ thuật:** Confusion Matrix phân tích False Positive/Negative
